# laptop-diagnostics

This is a simple Linux CLI tool that displays information about the system like battery health and the system serial number.
It is designed to be put on a Linux Live USB to diagnose a great number of devices efficiently.

## Installation

You can build this project using `cargo`

```bash
cargo install --git https://github.com/simon0302010/laptop-diagnostics.git
```

The binary should now be installed in ~/.cargo/bin/

You can also clone this GitHub repository and build it using cargo.
